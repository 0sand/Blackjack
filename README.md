# Blackjack

Peli on Blackjack korttipeli. Peli on tehty niin että se vastaa oikean casionon blackjackia mahdollisimman tarkasti. Pelissä on mallinettu korttipakka joka mahdollistaa että kehittynyt pelajaa hyötyy pelissä jos hän muistaa mitkä kortit on jo tullut. Koodissa löytyy jo mahdollisuus pelata useamalla pakalla mutta valinta ei ole vielä toteutettu vielä grafiseen käyttöliittymään. Peli on täysin toimiva mutta siinä ei ole vielä toteutettu mahdollisuus splittaukseen eikä panoksen tuplamiseen.
